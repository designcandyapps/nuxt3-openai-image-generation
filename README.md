## Setup

Make sure to install the dependencies:

```bash
yarn install
```
Copy `.env.example` to `.env`
```env
# Go https://beta.openai.com/account/api-keys to create
NUXT_OPENAI_API_KEY=openai_api_key
```

## Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
```
